CKANEXT-DEMO
============

CKAN 2.0 demo theme.

Installation
------------

    git clone https://github.com/okfn/ckanext-demo
    cd ckanext-demo
    pip install -e .
    pip install -r pip-requirements.txt

In the CKAN config file, enable the demo and dataset\_extent\_map
plugins (the latter is found in ckanext-spatial).
